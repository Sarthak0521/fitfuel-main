# 💪 FitFuel

> A full-stack fitness and nutrition platform designed to deliver personalized workout and diet experiences through a structured and scalable system.

FitFuel is a self-designed and developed web application that enables users to manage their fitness journey by combining workout planning, nutrition tracking, and personalized recommendations in a single platform.

The system is built with a focus on clean architecture, user-centric design, and real-world usability, making it suitable for both individual users and scalable fitness solutions.

---

## 🚀 About the Project

FitFuel was designed from scratch to solve a common problem — **lack of structured and personalized fitness guidance**.

Instead of relying on multiple apps, FitFuel provides:

* A centralized dashboard
* Structured workout plans
* Nutrition insights
* User-specific data tracking

The system also includes an **admin panel** to manage workouts, diet plans, and user-related content efficiently.

---

## 🧠 System Design Highlights

* 🔐 Authentication system using Supabase
* 👤 User profile management with personalized data
* 📊 Dashboard-driven architecture
* 🧑‍💼 Role-based admin panel
* 🗄️ Database structured with relational mapping (`auth.users → profiles`)
* ⚡ Real-time ready backend (extendable)

---

## 🚀 Features

* 🏋️ Workout management with categorized routines
* 🍽️ Nutrition plans with macro-level details
* 📊 Personalized dashboard for users
* 🔐 Secure authentication (email + OAuth)
* 🧑‍💼 Admin panel for content control
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** React.js, TypeScript, Tailwind CSS, Vite
* **Backend & Database:** Supabase
* **Authentication:** Supabase Auth
* **State Management:** Context API
